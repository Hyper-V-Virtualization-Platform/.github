## 🚀 Getting Started with Virtual Machines

*Prerequisites: A Windows Pro or Enterprise PC with hardware virtualization enabled in BIOS.*

1.  **Enable Hyper-V:** Go to Windows Features, enable Hyper-V, and restart your computer to activate this virtualization platform.
2.  **Launch Hyper-V Manager:** Open the virtual machine software from Start menu to begin creating and managing virtual machines.
3.  **Create Your First VM:** Use the New Virtual Machine wizard in this hypervisor technology to specify RAM, network, and storage for your virtual environment.
4.  **Install Operating System:** Mount an ISO file and install your chosen operating system within the virtual machine using this windows virtualization platform.
